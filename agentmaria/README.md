# AgentMaria - Project Management Dashboard

![Banner](https://img.shields.io/badge/Status-Development-orange)
![Tech Stack](https://img.shields.io/badge/Stack-React_|_Tailwind_v4_|_React_Flow-blue)

AgentMaria is a high-performance Project Management Dashboard and Workflow Editor. It allows users to build and simulate complex multi-agent workflows using a drag-and-drop node interface.

## ✨ Features

- **Project Grid**: Manage multiple projects with a sleek, interactive dashboard.
- **Node-Based Editor**: Drag and drop custom nodes (Planner, Developer, Researcher, etc.) to build workflows.
- **Real-Time Simulation**: Execute workflows with a topological simulation engine and visual data flow.
- **SaaS Dark Mode**: Premium UI designed with Tailwind CSS v4 and Lucide-Icons.
- **Local Persistence**: All workflows are saved automatically to your browser's storage.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
